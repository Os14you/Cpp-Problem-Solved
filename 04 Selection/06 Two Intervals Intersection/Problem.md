# Two Intervals Intersection

- Read 4 numbers represent 2 intervals and **print the intersection between them** , if none print -1.
#### 01 Example
```
    Input : 1 5 2 8        #[1,5] & [2,8]
    >> 2 , 5                 #The intersection is [2,5].
```
#### 02 Example
```
    Input : 1 5 6 10
    >> -1
```
