# Compare strings

- Read 2 strings, then output **YES** *if the first bigger than or equal to second*. *Otherwise*, output **NO** .
— **Don’t use < operator to compare strings. Use loops** .
#### 01 Example
```
    Input : Moustafa Saad
    >> YES
```
#### 02 Example
```
    Input : Mo Saad
    >> NO
```
